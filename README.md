# FizzBuzz

## Description

- Print every integer in range of 200,
**BUT**
- print Fizz if the integer is a multiple of 3 (1),
- print Buzz if the integer is a multiple of 5 (2),
- print FizzBuzz if (1) and (2)

## Install

```
git clone https://github.com/ZephOne/FizzBuzz.git
cd FizzBuzz
cargo run
```
